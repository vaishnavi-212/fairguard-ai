# FairGuard AI – Multi-Agent Fairness Copilot

## Problem
Hidden bias in automated decision systems can cause unfair hiring outcomes.

## Solution
FairGuard detects bias, scores fairness, flags disparities and supports responsible AI auditing.

## Features
- Bias Detection Engine
- Fairness Audit Score
- Risk Classification
- Explainable Bias Flagging
- Multi-Agent Governance Framework

## Tech Stack
Python  
Streamlit  
Pandas  
Google Gemini (planned integration)

## Demo Workflow
Upload CSV → Detect Bias → Get Fairness Audit Results

## USP
Unlike traditional bias tools that only diagnose fairness issues, FairGuard combines auditing, explainability and AI-assisted governance in one copilot.
